# 附录

## 状态码说明

| 状态码 | 说明                                |
| ------ | ----------------------------------- |
| -1     | 没有登录，需要验证                  |
| 1      | 成功                                |
| -2     | 业务失败, 错误原因请参见message字段 |
| -3     | 没有权限                            |
| -4     | 系统内部错误                        |

## 常见注意事项

1. 请求地址是什么？

> - API的访问路径为 ：[http://127.0.0.1:3000] + 请求路径

2.token的实效为多久？

> 为保证安全，token会在24h后实效，请定期更换token秘钥
